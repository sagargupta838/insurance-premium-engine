# insurance-premium-engine
Motor insurance premium calculation engine built with .NET Core - featuring IDV calculation, NCB discounts, GST computation and  multi-coverage pricing. Clean architecture with DI and async patterns.

A .NET Core backend service for motor insurance 
premium calculation, built to demonstrate 
clean architecture principles in an insurance 
domain context.

> Work in progress — actively being built

## What This Will Cover
- Motor insurance premium calculation (IDV based)
- NCB (No Claim Bonus) discount logic
- GST computation
- Multi-coverage add-on pricing
- Clean architecture — Controller/Service/Repository layers

## Tech Stack
- .NET Core 10
- SQL Server
- Docker
- Swagger / OpenAPI
